# Amphion demo (Hugging Face Spaces)

Paste a peptide → predicted antimicrobial probability, potency (MIC), hemolytic
probability, novelty, and the pipeline's pass/fail gate verdict.

## Run locally
```bash
.venv/Scripts/python app/app.py     # from the repo root
```

## Deploy to a free Hugging Face Space
1. Create a new **Gradio** Space.
2. Add these files to the Space repo:
   - `app.py` (this folder's `app.py`),
   - `requirements.txt` (this folder's),
   - the `src/amphion/` package,
   - the trained artifacts: `models/activity_clf.joblib`, `models/mic_reg.joblib`,
     `models/tox_clf.joblib`, and `data/processed/*.parquet` (needed for novelty lookup).
3. The metadata header above tells the Space to run `app.py`.

> **Model files note.** The repo git-ignores `models/*.joblib` (they're regenerated by the
> training entrypoints). For a Space you must either commit those three joblibs to the Space
> repo, or add a build step that trains them on startup. They're small (a few MB each).

> Predictions only — not validated results.
